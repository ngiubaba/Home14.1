## Электронная торговля
Консольная программа для добавления товаров в категории
### Функционал:
1. Реализован класс продукта
   * Реализован подкласс смартфоны
   * Реализован подкласс Трава газонная
2. Реализован класс категорий
   * Счетчик кол-ва продуктов
   * Счетчик категорий
3. Реализовано сложение продуктов, только при одинаковом их классе
4. Реализовано добавление продуктов, только относящихся к нашим категориям
5. Добавлен базовый абстрактный класс для всех продуктов BaseProduct.
6. Добавлен класс MixinPrintClass, выводящий информацию для отладки
7. Добавлено отображение средней цены всех продуктов в категории
8. Добавлена обработка исключений:
   * При 0 кол-ве товара
   * При отсутствии товаров в категории
   * При возможном делении на 0, при нахождении средней цены

### Тесты:
